## 📌 Overview

The rapid spread of misinformation on digital platforms has made it increasingly difficult to distinguish authentic news from fabricated content.

This project presents an intelligent **Fake News Detection System** that combines **Natural Language Processing (NLP)** and a **CNN-BiLSTM Deep Learning architecture** to classify news articles as **Real** or **Fake**.

The system supports **multilingual news analysis**, URL-based verification, and confidence-based predictions through an interactive Flask web application.

## 🏗️ System Workflow

1. User enters News Text or News URL
2. Content Extraction & Cleaning
3. NLP-Based Text Preprocessing
4. Feature Vector Generation
5. CNN Layer extracts local textual patterns
6. BiLSTM Layer captures contextual dependencies
7. Classification Model predicts Fake or Real
8. Confidence Score is generated
9. Results displayed through Flask Dashboard

---

## 🧠 Deep Learning Architecture

### CNN (Convolutional Neural Network)

* Extracts important keywords and local text patterns
* Learns semantic features from news articles

### BiLSTM (Bidirectional Long Short-Term Memory)

* Understands contextual relationships in both directions
* Captures long-range dependencies in news content

### Hybrid CNN-BiLSTM Model

* Combines feature extraction and sequence learning
* Improves classification performance for complex textual data
